## 💻 Technology Stack Details

### Frontend Framework
- **Next.js 16.2.6**: React framework with file-based routing, SSR, and automatic optimization
- **React 19**: Latest React with improved hooks and performance
- **TypeScript**: Full type safety with interfaces and strict mode

### Styling & Layout
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development
- **Responsive Design**: Mobile-first approach with breakpoints at 640px, 1024px
- **Custom Colors**: Green color palette representing nature theme
- **Framer Motion**: Smooth animations and transitions

### State Management
- **Zustand**: Lightweight state management for cart and orders
- **No Redux boilerplate**: Minimal setup, maximum efficiency
- **TypeScript Support**: Full type safety

### UI & UX
- **Lucide React**: 200+ professional icons
- **React Hot Toast**: Non-intrusive notifications
- **Next.js Image**: Automatic image optimization

### Build & Development
- **Turbopack**: Lightning-fast build tool
- **ESLint**: Code quality and consistency
- **PostCSS**: CSS processing and optimization
